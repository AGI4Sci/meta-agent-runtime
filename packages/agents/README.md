# Agent Adapter Packages

Each subdirectory here owns the migration of one source agent into the shared runtime framework.

- `smolagents/`
- `swe-agent/`
- `openhands/`
- `claude-code-sourcemap/`
- `cline/`
- `goose/`
- `ii-agent/`
- `opencode/`
- `pi-mono/`

These packages are intentionally isolated so different teams can port components in parallel without blocking one another.
